----------------Math.trunc()----------------

Math.trunc(x) returns the integer part of x

----------------Math.sign()----------------

Math.sign(x) returns -1 if number is negative,0 if number is null or 1 if number is positive

----------------Math.cbrt()----------------

Math.cbrt(x) returns the cube root of x

----------------Math.log2()----------------

Math.cbrt(x) returns the log value based on 2

----------------Math.log10()----------------

Math.cbrt(x) returns the log value based on 10
